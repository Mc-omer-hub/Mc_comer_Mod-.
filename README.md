# Mc_comer Mod 使用说明

## 简介

Mc_comer Mod 是一款 Cheat Engine 7.6 辅助工具，帮助你快速加载 Mod 文件并自动配置 CE 环境。

帮助我的世界开发者快速建立外挂环境,以便测试反作弊

---

## 下载

请先访问链接https://github.com/Mc-omer-hub/Mc_comer_Mod-./releases/tag/Mod
去下载最新的版本

---
## 登录

运行程序后，首先会看到登录页面：

1. 在密码框输入访问密码
2. 点击「登录」按钮进入主界面

---

## 主界面功能

登录成功后进入主界面，包含两个主要功能卡片：

### 1. 打开 Mod

用于加载 CT 文件并自动调用 Cheat Engine 打开。

- 点击「打开Mod」按钮
- 程序会自动查找已安装的 Cheat Engine
- 成功启动后会显示「已启动 CE」状态

> ⚠️ 需要先安装 CE 环境才能使用此功能

### 2. 安装 CE 环境

一键将 Cheat Engine 7.6 解压到桌面。

- 点击「安装CE环境」按钮
- 等待解压进度完成
- 安装成功后按钮会变为「已安装」状态

---

## 更新检查

程序启动时会自动检查更新：
- 如有新版本，会弹出更新提示窗口
- 点击「开始更新」可跳转到下载页面
- 点击「不更新」继续使用当前版本

---

## 常见问题

**Q: 提示"未找到 Cheat Engine.exe"怎么办？**  
A: 请先点击「安装CE环境」安装 Cheat Engine。

**Q: 忘记密码怎么办？**  
A: 请联系作者获取最新密码。

**Q: 安装 CE 环境后还是打不开 Mod？**  
A: 请检查 CE 是否正确安装到桌面，或尝试重新安装。
