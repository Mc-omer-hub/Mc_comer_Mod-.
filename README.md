# Erable_Mod 使用说明

## 简介

Erable_Mod 是一款基于 Cheat Engine 7.6 / python 辅助工具。

帮助我的世界开发者快速建立外挂环境,以便测试反作弊

---

## 下载

请先访问链接https://github.com/Mc-omer-hub/Mc_comer_Mod-./releases/tag/Mod
去下载最新的版本

---
## 登录

运行程序后，首先会看到登录页面：

1. 在密码框输入访问密码
2. 点击「登录」按钮进入主界面

OR
通过邮箱或扫码进行登录

---

---

## 更新检查

程序启动时会自动检查更新：
- 如有新版本，会弹出更新提示窗口
- 点击「开始更新」可跳转到下载页面
- 点击「不更新」继续使用当前版本
